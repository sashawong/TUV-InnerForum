# 轮胎法规论坛系统 - 项目需求表

## 1. 项目概述

### 1.1 项目名称
轮胎法规论坛系统

### 1.2 项目目标
为轮胎行业从业者提供一个专业的法规信息分享和交流平台，支持用户发布法规更新、技术问题求助、经验分享等内容。

### 1.3 目标用户
- 轮胎行业工程师
- 法规合规专员
- 质量管理人员
- 技术研发人员

## 2. 功能需求

### 2.1 用户管理

#### 2.1.1 用户注册和登录
- 支持用户名和密码登录
- 密码使用 bcrypt 加密存储
- JWT token 身份验证
- 支持用户登出功能

#### 2.1.2 用户角色
- **普通用户**：可以浏览、发帖、回复、收藏
- **管理员**：拥有所有权限，可以管理后台

### 2.2 帖子管理

#### 2.2.1 帖子类型
- **分享**：分享法规更新、技术经验等
- **求助**：提出技术问题，寻求帮助

#### 2.2.2 帖子创建
- 支持填写标题和内容
- 支持选择帖子类型（分享/求助）
- 支持添加标签（如：EU, UNECE, MAS等）
- 支持上传图片（最多10张）
- 支持上传附件（最多10个）
- 支持中文文件名的附件上传

#### 2.2.3 帖子浏览
- 帖子列表展示
- 支持按标题和内容搜索
- 显示帖子标题、内容、作者、创建时间
- 显示点赞数、踩数、回复数
- 显示帖子类型标签
- 显示自定义标签（彩色标签）
- 支持置顶帖子

#### 2.2.4 帖子详情
- 显示完整帖子内容
- 显示帖子图片（支持预览）
- 显示帖子附件（支持下载）
- 显示所有回复
- 显示点赞数、踩数

#### 2.2.5 帖子删除
- 帖子作者可以删除自己的帖子
- 管理员可以删除任何帖子
- 删除帖子时同时删除相关的回复、点赞、收藏、图片、附件

### 2.3 回复管理

#### 2.3.1 回复创建
- 支持填写回复内容
- 支持上传图片（最多10张）
- 支持上传附件（最多10个）
- 支持嵌套回复（回复其他回复）

#### 2.3.2 回复展示
- 显示回复内容、作者、创建时间
- 显示回复图片
- 显示回复附件
- 显示点赞数、踩数
- 支持嵌套回复展示

### 2.4 互动功能

#### 2.4.1 点赞/踩
- 支持对帖子点赞/踩
- 支持对回复点赞/踩
- 每个用户只能对同一帖子/回复操作一次
- 实时更新点赞数、踩数

#### 2.4.2 收藏
- 支持收藏帖子
- 在用户中心查看收藏列表

### 2.5 用户中心

#### 2.5.1 账号设置
- 修改用户信息
- 修改密码

#### 2.5.2 我的收藏
- 查看收藏的帖子列表
- 支持跳转到帖子详情

#### 2.5.3 历史记录
- 查看浏览历史

#### 2.5.4 消息
- 查看系统消息

### 2.6 后台管理（管理员）

#### 2.6.1 用户管理
- 查看所有用户
- 管理用户角色

#### 2.6.2 帖子管理
- 查看所有帖子
- 删除不当帖子

### 2.7 主题设置

#### 2.7.1 主题切换
- 支持亮色主题
- 支持暗色主题
- 主题切换实时生效

#### 2.7.2 语言设置
- 支持中文
- 支持英文

## 3. 非功能需求

### 3.1 性能需求
- 页面加载时间 < 2秒
- 图片上传支持大文件（建议 < 10MB）
- 附件上传支持大文件（建议 < 50MB）

### 3.2 安全需求
- 密码加密存储
- JWT token 身份验证
- 文件上传安全检查
- 防止 SQL 注入（使用 LowDB，不涉及 SQL）

### 3.3 可用性需求
- 响应式设计，支持桌面端和移动端
- 友好的用户界面
- 清晰的错误提示
- 支持中文和英文界面

### 3.4 兼容性需求
- 支持主流浏览器（Chrome, Firefox, Edge, Safari）
- 支持最新版本的浏览器

- 用户名

- 内容
- 作者ID

- 创建时间

- 图片路径
- 创建时间

### 4.5 附件数据
- 附件ID
- 帖子ID/回复ID

- 创建时间

### 4.6 点赞数据

- 帖子ID/回复ID
- 用户ID

- 创建时间

### 4.7 收藏数据
- 收藏ID
- 帖子ID
- 用户ID
- 创建时间

## 5. 界面需求

### 5.1 主页
- 顶部导航栏（搜索框、用户信息）
- 左侧边栏（菜单导航）
- 主内容区（帖子列表）
- 新建帖子按钮

### 5.2 帖子详情页
- 帖子信息展示区
- 图片展示区（支持预览）
- 附件下载区
- 回复列表区
- 回复输入区

### 5.3 用户中心
- 标签页导航（账号设置、我的收藏、历史记录、消息）

### 5.4 后台管理页
- 用户管理表格
- 帖子管理表格
- 操作按钮（编辑、删除等）

## 6. 文件存储需求

### 6.1 图片存储
- 存储路径：`backend/uploads/`
- 文件命名：`{timestamp}-{random}.{ext}`
- 支持格式：jpg, jpeg, png, gif, webp

### 6.2 附件存储
- 存储路径：`backend/uploads/`
- 文件命名：`{timestamp}-{random}.{ext}`
- 支持格式：pdf, doc, docx, xls, xlsx, txt, csv, zip等

### 6.3 文件访问
- 通过静态文件服务访问
- URL 格式：`/uploads/{filename}`

## 7. 优先级

### 7.1 高优先级（P0）
- 用户注册和登录
- 帖子创建和浏览
- 回复功能
- 基本的互动功能（点赞/踩）

### 7.2 中优先级（P1）
- 图片和附件上传
- 标签系统
- 收藏功能
- 用户中心

### 7.3 低优先级（P2）
- 历史记录
- 消息功能
- 主题切换
- 后台管理
